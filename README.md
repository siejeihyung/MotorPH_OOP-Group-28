<h3>File Overview</h3>
<p>This project focuses heavily on the practical application of <b>Encapsulation</b> and <b>Abstraction</b>:</p>

<ul style="list-style-type: square; padding-left: 25px;">
  <li>
    <b>Encapsulation:</b> Applied in <code>Employee.java</code> using <code>this</code> references and <b>getters</b> to protect object states.
  </li>
  <li>
    <b>Abstraction:</b> The <code>grossPay.java</code> interface defines the structural requirements for <b>Weekly Gross Salary</b> computations.
  </li>
</ul>

<p>For a clear interpretation of how these are implemented, refer to these specific files:</p>
